1

Recount the SKU across all bins, overflow, and staging areas. Check open pick/pack orders for undeducted shipments. Review transfer logs from the past 48 hours. If the shortage persists, submit an Inventory Adjustment Request with variance amount and root-cause notes.

Variances >5% of SKU value need manager sign-off before processing.

Receiving Issues
01Shipment arrived with damaged goods
1
Do NOT receive
2
Photo + document
3
Move to DMG-HOLD
4
File exception

Do not receive damaged items into available inventory. Photograph damage, note carrier, BOL#, and PO#. Segregate to DMG-HOLD. File a receiving exception and notify Procurement within 24 hours.

Carrier claims must be filed within 5 business days. Photograph everything immediately.

Transfer Errors
01Transfer sent but not confirmed at destination
1
Check "Shipped" status
2
Call receiving site
3
Complete receipt
4
Flag In Dispute

Verify the transfer shows "Shipped" at origin. Call the receiving facility — goods may have arrived unscanned. If on-site, have them receipt now. If missing in transit, flag In Dispute and loop in Logistics.

02Items transferred to the wrong location code
1
Find at wrong code
2
Move & re-scan
3
Submit correction
4
Investigate if missing

Locate items at the wrong code. Move to the correct bin and scan in. Submit a location correction with old and new codes. If items aren't where expected, open a discrepancy investigation.

System & Data
01Inventory sync failed between systems
1
Pull error logs
2
Don't force-match
3
Ticket to IT
4
Override if blocking

Check the integration dashboard for error logs — grab the timestamp and error code. Do not manually adjust quantities to force a match. Ticket IT with the details and affected SKUs. If operations are blocked, request a manual override with IT approval.

Integration Dashboard IT Ticket Portal
02Duplicate SKU or lot number created
1
ID both records
2
Find source of truth
3
Don't delete — merge
4
Consolidate physical

Note both SKU/lot numbers, quantities, and locations. The first-created record is usually the source of truth. Do not delete either — submit a Data Merge Request to IT. Consolidate physical stock under one location until the merge completes.

03Manual inventory adjustment needed
1
Document root cause
2
Fill Adjustment Form
3
Get approval
4
Process + attach

Select a root-cause category (shrinkage, damage, data error, etc.). Complete the form with SKU, current qty, proposed qty, and justification. Approval: team lead for <$500, manager for ≥$500. Process and attach the signed form to the transaction record.

All adjustments are audited quarterly. Incomplete documentation will be flagged.
