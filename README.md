Static internal knowledge base site for inventory procedures, designed for GitHub Pages hosting.

```text
.

`-- assets/
    |-- css/
    |   `-- styles.css
    |-- js/
    |   `-- main.js
    `-- img/
        `-- amlongroup-logo.svg
```

## Local Preview

Open `index.html` directly in a browser, or run a local static server if you prefer.

## Deploy to GitHub Pages

1. Push this repository to GitHub.
2. In repository settings, open **Pages**.
3. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
4. Select the `main` branch and `/ (root)` folder.
5. Save and wait for deployment.

Your site will publish from the root `index.html` and static files in `assets/`.
